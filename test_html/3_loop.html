Loops : for while

동일한 코드를 반복해서 실행하고 싶을때 유용

반복이란? 일상생활에서 똑같은 행동에 패턴을 유지하는것을 반복이라한다
하지만 개발에서는 지정된 범위만큼 실행하는 것을 반복이라 한다.
- for문을 사용하는 경우: 횟수가 정해져 있을 때 범위가 명확할때 최적화
- while문 : 반복 횟수가 정해져있지 않고, 어떤 조건이 참인 동안 계속 실행하고 싶을때
조건검사 조건먼저확인-> 참이면 실행 -> 다시조건검사
- do-while 문: 일단 한 번 실행하고 보자 최소한번실행
일단 실행 -> 그다음 조건검사 -> 참이면 반복

cors : Cross-Origin Resource Sharing

교차 출처 리소스 공유

- 서로 다른 출처를 가진 서버끼리 자원을 주고 받을 수 있게 허용하는 체계 - 브라우저는 기본적으로 동일 출처정책(Same-Origin Policy)을 따릅니다 - 내 출처와 다른 곳에서 온 요청은 보안상 위험할 수 있으니 차단하겠다. 그래서 CORS라는 약속을 통해 이 출처는 내가 허용할게!

java: http://localhost:8080
react, nextjs: http://localhost:5174
