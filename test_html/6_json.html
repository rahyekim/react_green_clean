JSON을 사용하는이유

컴퓨터간의 데이터를 쉽게 전송하고 저장 JSON문자열을 자바스크립트 객체로 변환하는 내장함수 JSON.parse()

이미 내 컴퓨터 메모리에 완벽하게 로드된 순수한 텍스트 글자(String)를 자바스크립트 객체로 바꿀 때 인터넷 연결도 필요 없고, 기다릴 필요도 없이 즉시(동기적으로) 실행됩니다. (cf) 변수.json() :인터넷에서 '다운로드 중인 데이터'를 바꿀 때 response.json()은 다음 2가지 일을 한 번에 해줌 1.네트워크로 들어오는 데이터가 끝날 때까지 다 다운로드한다. (비동기적await가 필요한 이유) 2. 다운로드가 끝난 데이터를 내부적으로 JSON.parse() 같은 함수를 돌려서 자바스크립트 객체로 변환해 준다.

JSON.parse(): 이미 내 손에 있는 JSON형식의 문자열: 순수 텍스트 파싱 기능만 수행

response.json() : fetch로 막 받아온 네트워크 응답 객체

자바스크립트 객체를 JSON문자열로 변환하는 내장함수 JSON.stringify()
