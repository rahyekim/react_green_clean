         ㄴ 원, 각도, 그래프, 애니메이션에서 가끔 사용
    Math.SQRT2 [루트2, 약 1,414]: 2의 제곱근 1:1 비율인 정사각형의 대각선 길이를 구할때 사용

Math.random()

        📌 Math.random() : 0 이상 ~ 1 미만(소수점) 무작위생성
        
        ⚠️절대1은 안나옴!

        Math.random() * 10 👉 0~ 9.99999 ✔️ 소수점생김
        Math.floor(Math.random() * 10) +1  👉 1 ~ 10
        Math.floor(Math.random() * 10) 👉 0 ~ 9
        Math.floor(Math.random() * 11) + 5 -> // 5~15

        💬 배열에서 랜덤 뽑기 (랜덤 배너)

⭐Math.floor(Math.random() * 개수) + (시작값)⭐

개수 = 배열변수.length

Math.sign(x)

숫자의 부호(sign)가 무엇인지 딱잘라 말해주는 도구 정확히 얼마인지 중요하지 않고 -음수 0 +양수 만 확인하고싶을때 결과값은 항상 -1, 0 , 1 세가지중하나만 나옴 Math.sign(-5) -> -1 게임에서 캐릭터가 오른쪽or왼쪽 가는지 확인할때
