- 2진수(Binary): 컴퓨터의 기본언어 컴퓨터 내부 회로는 전류가 흐르느냐(1) 흐르지 않느냐(0)로 동작.=> 1bit: 0or1 딱 하나넣을수있는 제일작은상자한칸 0과 1 단 두개의 숫자만 사용하는것이 2진수 특징: 모든 정보는 2진수로 변환되어 처리됨 10진수 100이 이진수 1100100 -> 100을 2로 나누면서 나머지를 아래에서 위로 읽음 -8진수(Octal) : 2진수를 3자리씩 묶은 것 -16진수 :0부터 15까지 총 16개의 숫자를 한 자리에 표현(0~9,A~F), 2진수를 "4자리씩" 예쁘게 묶은 것. 2진수: 1111 1111 (길고 읽기 힘듦 1byte:한글자) => 딱 두 글자(00 ~ FF)로 완벽하게 요약 

                    1 Byte 상자 안에는 2진수 8자리(00000000 ~ 11111111):256가지 다른 신호 만들수잇음.
                    - 2진수 4 Bit = 16진수 1자리 (예: F)
                    - 2진수 8 Bit (1 Byte) = 16진수 2자리 (예: FF)
