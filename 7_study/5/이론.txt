⭐ 네트워크 

🟦 OSPF ( Open Short Path First)
- 대표적인 링크상태 라우팅 프로토콜
- 단일 자율시스템 내에서 라우팅 정보를 배포
- 모든 대상에 도달하기 위한 최단 경로 구축 
- 최단경로는 Dijkstra알고리즘을 사용하여 계산

⭐조인관련

🟥 Theta Join, θ, 세타조인
일반적인 조인: 조인에 참여하는 두 릴레이션 속성 값을 비교하여
조건을 만족한 튜플만 반환
가장보편적이고 포괄( > < != ) 모든 비교 연산자 사용 가능
이런것들로만 그 조건이 참인 데이터(튜플)만 엮어냄

🟦 동등 조인(Equi Join): 
세타 조인 중에서 =만 사용하는 조인
조건이 정확하게 = 등호로 일치하는 결과를 리턴 
위에 세타조인에서 비교연산자를 오직 등호만 사용하는 조인 

🟥 자연조인
동등조인에서 조인에 참여한 속성이 두번 나오지 않도록 중복된
속성을 제어한 결과...


⭐패턴관련 

🟦 Absctact Factory 
- 구체적인 클래스에 의존하지 않고 서로 연관되거나
  의존적인 객체들의 조합을 만드는 인터페이스를 제공하는 패턴
- 연관성이 있는 객체군이 여러개 있을 경우
  이들을 묶어 추상화 하고 어떤 구체적인 상황이 주어지면
  팩토리 객체에서 집합으로 묶은 객체 군을 구현화하는 생성 패턴

- 관련성이 있는 여러 종류의 객체를 일관된 방식으로 생성하는경우 유용
= kit 라고도 부른다

🟦 Rootkit 루트킷
인터넷의 공격자의 존재를 숨기면서 
이공격자에게 시스템에 대한 무제한 접근권한을 부여
해커가 자신의 존재를 숨기면서 허가되지 않은 컴퓨터나 소프트웨어에 
접근할 수 있도록 설계된 도구
일반적으로 펌웨어 가상화 계층등의 다양한 시스템영역에서 작동
운영체제에서 시스템콜을 해킹하여 악성코드의 실행여부를 숨켜 안티바이러스
탐지를 우회한다.
